Brawl Shooters

Brawl Shooters is a top-down roguelike shooter developed in Unity using C#. The project focuses on implementing scalable gameplay systems such as enemy AI behaviour, wave spawning, player progression and weapon upgrades.

Gameplay Overview

The player must survive waves of zombies inside an arena. Each wave increases in difficulty by increasing the number of enemies. Every five waves a boss enemy spawns with significantly higher health.

  • Wave-based survival gameplay
  • Boss encounter every 5 waves
  • XP orbs dropped by enemies
  • Weapon upgrades when leveling up

Technical Systems

  • Wave spawning manager
  • Enemy AI chase behaviour
  • XP collection and leveling system
  • Weapon upgrade system
  • HUD elements for health, XP and waves

Technical Challenges

One issue during development was enemies spawning too close to the player. Early versions occasionally spawned enemies directly on top of the player, causing unavoidable damage.

This was fixed by adding a minimum spawn distance check when generating enemy spawn positions.

Another challenge was maintaining performance during large waves. Instead of using complex NavMesh pathfinding, a lightweight movement system was implemented where enemies directly move toward the player. Since the arena contains no obstacles, this approach significantly improved performance while keeping the behaviour believable.
