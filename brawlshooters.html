Brawl Shooters

Brawl Shooters is a top-down wave-based action game developed in Unity. The project focuses on designing scalable gameplay systems including combat, enemy AI behaviour, and player progression.

Gameplay Systems Implemented

  • Player movement and shooting mechanics
  • Enemy AI behaviour and targeting
  • Wave spawning system
  • Player progression and upgrade system
  • Gameplay UI feedback

Technical Focus

The core focus of the project was gameplay programming and system scalability. The wave spawning system dynamically increases enemy difficulty over time, creating an adaptive gameplay loop that challenges the player as they survive longer.
