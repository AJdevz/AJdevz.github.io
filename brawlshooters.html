Brawl Shooters

Brawl Shooters is a top-down wave-based action game developed in Unity. The goal of the project was to design and implement scalable gameplay systems for combat and enemy behaviour.

Gameplay Systems Implemented

  • Player movement and shooting mechanics
  • Enemy AI behaviour
  • Wave spawning system
  • Player progression and upgrades
  • Gameplay UI and feedback

Technical Focus

The project focuses on gameplay programming and system design. The wave spawning system dynamically increases enemy difficulty as the player survives longer, allowing for scalable gameplay progression.
