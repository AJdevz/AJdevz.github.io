Featured Projects

Convergence

Convergence is an experimental gameplay systems project built in Unity. It expands on the original Brawl Shooters prototype by introducing multiple gameplay modes and adaptive enemy behaviours. The project currently contains two main gameplay modes: • Classic Mode — traditional wave-based combat inspired by Brawl Shooters • Merge Mode — enemies dynamically merge together to create stronger variants, forcing the player to adapt strategies in real-time. The merging system required designing custom logic to detect enemy proximity, combine enemy stats and dynamically update behaviour and difficulty.

Brawl Shooters

Brawl Shooters is a solo university project where I implemented the core gameplay systems for a top-down wave-based shooter. The goal of the project was to build scalable gameplay systems that could be easily expanded during development. Key systems implemented include: • Player movement and combat mechanics • Enemy AI behaviours • Wave spawning system with increasing difficulty • Player progression and upgrade mechanics • Gameplay feedback and UI integration
